# cyril_lexicon_mss
This repository contains the list of manuscripts of Cyril Lexicon studied within the framework of of the PRIN2022 project PatriarX, combining the results of the recensio carried out by the Messina and Basilicata Research Units with those of the recensio conducted by Mark Naoumides, brought to light by the Verona Research Unit.
The list includes basic information for each manuscript (place of conservation, date, folia containing Cyril's Lexicon), specifying whether Naoumides was aware of the manuscript and, if so, to which family he assigned it.


## Header description
| Column Name           | Description                                                                                                                                                                                                                                                   |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| city                  | The modern city where the holding institution (library, museum, or archive) is located. This is the first-level geographic identifier for the manuscript's current location.                                                                                  |
| library               | The name of the institution (library, museum, or archive) that holds the manuscript. This is the second-level identifier for the manuscript's current location.                                                                                                               |
| msIdentifier          | The local shelfmark assigned by the holding institution to uniquely identify this specific manuscript within its collection.                                                                                                                           |
| msItem                | Specifies the location of the Cyril Lexicon within the manuscript.                                                                                                               
| extent                | A general statement about the portion of the manuscript the record describes.                                                                                                               
| Diktyon               | A unique and persistent identifier for the manuscript record in the Pinakes database (https://pinakes.irht.cnrs.fr), the principal portal for Greek manuscripts. This number is a critical link for interoperability and further research.                                                                                                                                                                |                                  
| origDate_notBefore    | The earliest possible year the manuscript (or the specific item) could have been produced, according to scholarly dating. It provides a terminus post quem.                                                                                                   |
| origDate_notAfter     | The latest possible year the manuscript (or the specific item) could have been produced, providing a terminus ante quem.                                                                                                                                            |
| affiliation_naoumides | A specialized note, that clarifies the manuscript's attribution according to Mark Naoumides. "NOT CYR." indicates the item is not the Cyril Lexicon.                                                                                                                                                               |   
| note                  | A general-purpose field for any additional comments or observations about the manuscript or the record itself.        |
| studied               | A project-specific flag indicating whether the manuscript has been examined or fully processed ("yes") or remains to be studied ("no").                                                                                                                                                                                                                                                |
