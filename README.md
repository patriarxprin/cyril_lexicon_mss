# cyril_lexicon_mss
A list of manuscripts of Cyril Lexicon.



## Header description
| Column Name           | Description                                                                                                                                                                                                                                                   |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| city                  | The modern city where the holding institution (library, museum, or archive) is located. This is the first-level geographic identifier for the manuscript's current location.                                                                                  |
| library               | The name of the institution or collection that holds the manuscript. This is the second-level identifier for the manuscript's current location.                                                                                                               |
| msIdentifier          | The local shelfmark or call number assigned by the holding library to uniquely identify this specific manuscript within its collection. The value "olim" indicates a former shelfmark.                                                                        |
| msItem                | Specifies the location of the Cyril Lexicon (or related text) within the manuscript. This is crucial for codices that are composite or contain multiple works. For example, "ff. 1r-125v" means the text begins on folio 1 recto and ends on folio 125 verso. |
| extent                | A general statement about the portion of the manuscript the record describes. In your data, it is mostly blank or uses a slash ("/") to imply the entire manuscript is the focus, but it could also specify "ff. 1r-179v".                                    |
| Diktyon               | A unique and persistent identifier for the manuscript record in the Pinakes database (https://pinakes.irht.cnrs.fr), the principal portal for Greek manuscripts. This number is a critical link for interoperability and further research.                    |
| origDate_notBefore    | The earliest possible year the manuscript (or the specific item) could have been produced, according to scholarly dating. It provides a terminus post quem.                                                                                                   |
| origDate_notAfter     | The latest possible year the manuscript (or the specific item) could have been produced, providing a terminus ante quem.                                                                                                                                      |
| affiliation_naoumides | A specialized note, likely referencing the work of scholar Naoumides, that clarifies the text's attribution. "NOT CYR." indicates the item is not the Cyril Lexicon but a similar work (a note in the data points to Zonaras).                                |
| note                  | A general-purpose field for any additional comments or observations about the manuscript or the record itself.                                                                                                                                                |
| studied               | A project-specific flag indicating whether the manuscript has been examined or fully processed ("yes") or remains to be studied ("no"). This is a workflow management field.                                                                                  |
